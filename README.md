# 100xengineer

An experimental repo that gives an agent access to compute and coding resources.

## Tech Stack

- [React frontend](#react-frontend)
- Express Backend
- LangchainJS agent library
- gpt-3.5 as the model to start with
- k8s for giving agent resources
